# 📊 Calectra Dashboard - HTML Version

## ✨ What You Have

A **single-file HTML dashboard** that converts your React app into a standalone HTML application. No build process needed!

### Features
- ✅ Flood View dual-axis chart (RE Generation vs Battery Storage)
- ✅ Interactive LMP Analysis (Duck Curve with filters)
- ✅ Same dark mode design as React version
- ✅ Same data visualization and metrics
- ✅ Works offline (no Node.js or npm required)
- ✅ Can be deployed anywhere (static file)

## 🚀 HOW TO USE (Super Simple!)

### Option 1: Run Locally (Recommended)
1. Keep the file structure as-is:
   ```
   Calectra/
   ├── dashboard.html              ← Open this file
   └── public/
       └── data/
           └── dashboard_data.json  ← Must exist (generated by process_data.py)
   ```

2. **Open the HTML file in your browser:**
   - Double-click `dashboard.html` 
   - OR right-click → Open with → Browser

3. Dashboard loads instantly! ✨

### Option 2: Run with a Local Server
If you get CORS errors (unlikely), run a simple server:

**Windows (PowerShell):**
```powershell
cd "c:\Users\dlseh\Desktop\Calectra Github\Calectra"
python -m http.server 8000
# Then open: http://localhost:8000/dashboard.html
```

**Mac/Linux:**
```bash
cd ~/Desktop/Calectra
python -m http.server 8000
# Then open: http://localhost:8000/dashboard.html
```

## 🌐 Deploying to the Web

### Option 1: GitHub Pages (Free)
1. Create a GitHub repo
2. Add these files:
   - `dashboard.html`
   - `public/data/dashboard_data.json`
3. Go to Repo Settings → Pages → Select `main` branch
4. Your dashboard is live at: `your-username.github.io/repo-name/dashboard.html`

### Option 2: Netlify (Free)
1. Drag & drop `dashboard.html` onto https://drop.netlify.com/
2. Your dashboard gets a live URL instantly ✨

### Option 3: Your Web Server
Simply upload:
- `dashboard.html`
- `public/data/dashboard_data.json` (same relative path)

## ⚙️ Customization

### Change Colors
Edit the CSS variables at the top of the HTML file (~line 15):
```css
:root {
    --primary: #00D9FF;      /* Cyan - change to any color */
    --secondary: #FF6B35;    /* Orange - change to any color */
    --dark: #1a1f2e;
    --darker: #0f1419;
}
```

### Add Your Logo
Replace the header title with an image:
```html
<h1 class="header-title">
    <img src="your-logo.png" alt="Calectra" style="height: 50px;">
</h1>
```

### Change Company Name
Search for "Calectra" in the HTML and replace with your company name.

## 🔧 Troubleshooting

### "Cannot load data" Error
1. Check that `public/data/dashboard_data.json` exists
2. Run: `python process_data.py`
3. Refresh browser (Ctrl+R)

### Charts not showing
- Clear browser cache (Ctrl+Shift+Delete)
- Try incognito mode
- Check browser console for errors (F12)

### CORS Error (unlikely)
Run a simple Python server instead of opening the file directly (see above).

### Data not updating after running process_data.py
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Clear cache and reload
